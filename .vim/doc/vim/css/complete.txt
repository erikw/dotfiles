*vim-css-complete*


Css Code Completion
*******************

Css code completion uses the standard Vim code completion mechanism
(|vim-code-completion|) like so:

>

   bo<Ctrl-X><Ctrl-U>

   body {
     font-<Ctr-X><Ctrl-U>

   body {
     font-family: sa<Ctrl-X><Ctrl-U>

   body {
     font-family: sans-serif;
   ...

<
